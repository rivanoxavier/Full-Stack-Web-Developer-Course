# Full-Stack-Web-Developer-Course
This repository contains my progress and projects from Angela Yu’s Complete Full-Stack Web Development Bootcamp (Udemy).

The goal of this repository is to:
	•	Build a strong foundation in full-stack web development
	•	Practice concepts through hands-on projects
	•	Track learning progress consistently
	•	Maintain a portfolio for future internships and job opportunities
